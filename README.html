AIM_HUST-2022

华中科技大学狼牙战队2022赛季步兵自瞄程序。

Author:周倍进

简介

  1. 硬件配置

硬件上我们使用大恒相机+5mm焦距镜头,miniPC使用的是NVIDIA Jetson TX2

  1. 功能
  • 自瞄
  • 反陀螺
  • 能量机关
  1. 部署需要调节的参数有
  • 传统视觉相机参数，如曝光，增益等
  • 大恒相机ID
  • 相机内参数
  • 相机镜头到枪管的平移向量
  • USE转串口设备名字
  • 线程延时
  • 传统视觉识别参数，如二值化阈值,灯条装甲板几何约束参数
  • 相机到枪管的旋转角度yaw_bias和pitch_bias(相机安装不可能完全与枪管无旋转)
  • 发射延迟
  • 卡尔曼滤波参数，如测量噪声，过程噪声
  • 强跟踪滤波器参数

环境配置

  • Ubuntu 18.04 LTS
  • gcc 7.5.0
  • OpenCV 4.5.1
  • Eigen非模板类
  • Ceres
  • Sophus
  • LCM
  • Glog
  • Jsoncpp
